## Context (from L3)

post-tool-use.sh is a single-line bash script wired to Claude Code's PostToolUse lifecycle event by hooks.json. After every tool invocation by the agent, Claude Code execs this script (with a 5s timeout) and consumes its stdout as a hookSpecificOutput JSON object. The script's only job is to emit a fixed JSON payload via `jq -n` whose `additionalContext` string nudges the agent toward calling `/learn` at completion boundaries and `/prepare` at start-of-work boundaries. The script takes no input from stdin, makes no decisions, and produces the same output on every invocation; freshness/relevance of the reminder is the agent's problem, not the script's. `set -euo pipefail` ensures any jq failure causes a non-zero exit (rather than silently emitting partial JSON), and the shebang pins bash so the pipefail/strictness semantics are portable.
