## Context (from L3)

E15 c4 skill is the markdown skill body Claude Code loads when the user invokes `/c4 <sub-action>`. Unlike its sibling skills, it does NOT shell out to the engram CLI binary — its body instructs the agent to use `targ c4-*` build/audit/registry/render targets and to edit files under `architecture/c4/` directly. The properties below are behavioral guarantees the skill text imposes on the agent: dispatch rules, conflict-resolution discipline, propagation discipline, ID/format invariants, and untested-property handling. Tested rows point at the pressure-test markdown captures under `skills/c4/tests/`; rules with no pressure-test capture are marked **⚠ UNTESTED**.
